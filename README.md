# 💣 扫雷游戏 (Mine Sweeper)

一个使用 HTML/CSS/JavaScript 开发的网页版扫雷游戏。

## 🎮 游戏玩法

- **左键点击**：翻开方块
- **右键点击**：标记/取消标记地雷
- **目标**：找出所有非地雷的方块即可获胜

## 🏆 难度选择

| 难度 | 格子数 | 地雷数 |
|------|--------|--------|
| 简单 | 9×9 | 10 |
| 中等 | 16×16 | 40 |
| 困难 | 30×16 | 99 |

## 🚀 运行方式

直接在浏览器中打开 `index.html` 即可游玩。

## 🔧 技术栈

- HTML5
- CSS3
- JavaScript (原生)
